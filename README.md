# CorineGen - AI 图像生成器

基于 ComfyUI 的 React Web 应用程序，提供美观的黑紫色神秘主题界面，用于生成 AI 图像。

## 功能特性

- 🎨 黑紫色神秘感主题界面
- ✍️ 自定义提示词输入
- 🔢 批次数量选择（1, 2, 4）
- 📐 图像比例选择（Square, Portrait, Landscape）
- 🎲 种子模式（随机、固定、首次固定）
- 📊 实时进度反馈
- 🖼️ 图像预览和下载

## 前置要求

- Node.js 16+
- ComfyUI 运行在 `http://127.0.0.1:8188`

## 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 确保 ComfyUI 正在运行并可访问 `http://127.0.0.1:8188`

3. 启动开发服务器：
```bash
npm run dev
```

4. 在浏览器中打开 `http://localhost:3000`

## 使用说明

1. **输入提示词**：在文本框中输入您想要生成的图像描述
2. **选择批次数量**：选择一次生成 1、2 或 4 张图像
3. **选择图像比例**：
   - Square (1:1) - 1024x1024
   - Portrait (9:16) - 720x1280
   - Landscape (16:9) - 1280x720
4. **选择种子模式**：
   - 随机：每次生成使用新的随机种子
   - 固定：使用指定的种子编号
   - 首次固定：第一次随机生成，后续使用相同种子
5. **点击生成按钮**：观看进度条并等待图像生成完成
6. **下载图像**：生成完成后可以预览和下载图像

## 构建生产版本

```bash
npm run build
```

构建的文件将输出到 `dist` 目录。

## 技术栈

- React 18
- Vite
- ComfyUI API

## 注意事项

- 确保 ComfyUI 工作流文件 `CorineGen.json` 在项目根目录
- 如果 ComfyUI 运行在不同的地址，请修改 `src/App.jsx` 中的 `COMFYUI_API` 常量
